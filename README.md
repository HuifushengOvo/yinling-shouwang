# 银龄守望 - 老年数字生活互助站

高校社区老年数字生活互助站智能服务匹配系统，基于 HarmonyOS ArkTS 开发。

## 项目简介

解决老龄化社会面临的数字鸿沟问题，通过高校志愿者资源与老年用户需求对接，打造轻量可信的适老服务匹配平台。

## 技术栈

- HarmonyOS + ArkTS
- 三页面路由（Index / VoicePage / ResultPage）
- ForEach + Builder 组件模式
- 语音识别交互流程设计

## 页面功能

| 页面 | 功能 |
|------|------|
| Index | 首页服务选择（微信使用、医院挂号、医保社保、打车导航） |
| VoicePage | 3 秒语音录音、倒计时交互、AI 边缘盒对接 |
| ResultPage | 志愿者匹配结果展示、服务预约 |
